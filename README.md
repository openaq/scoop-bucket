## Scoop bucket for @openaq CLI binaries, powered by @goreleaser

- **Add the bucket**

    ```bash
    scoop bucket add openaq-bucket https://github.com/openaq/scoop-bucket
    ```

- **Install**

    ```bash
    scoop install openaq-cli
    ```
